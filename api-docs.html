Mock Media API Documentation

Manage movies, series, and songs via REST endpoints.

Endpoints

Movies

  • GET /movies - List all movies
  • GET /movies/name - Get a movie by name(Capital Letter for first letter of words between spaces)
  • POST /movies - Add a movie. Send JSON body: 
    {
    "name": "string",
    "year": "number",
    "restriction": "string",
    "hours": "string",
    "rating": "string"
}
  • PUT /movies/name - Update a movie. Send JSON body with fields to update. Send JSON body: 
    {
    "year": "number",
    "restriction": "string",
    "hours": "string",
    "rating": "string"
}
  • DELETE /movies/name - Delete a movie
